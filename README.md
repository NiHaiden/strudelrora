# Strudelrora

This image is based off of Aurora-DX and installs Steam and 1Password for myself. This image is mostly intended to be consumed by myself, but feel free to use it.

Name is based on Apfelstrudel, yes I am very creative.

## Variants

- `latest`: based on `ghcr.io/ublue-os/aurora-dx:stable`
- `beta`: based on `ghcr.io/ublue-os/aurora-dx:beta`

Strudelrora ships a custom fastfetch config at `/usr/share/ublue-os/fastfetch.jsonc`.
